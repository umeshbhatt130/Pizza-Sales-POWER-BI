## ğŸ“Š Key KPIs (Metrics Tracked)
The following KPIs were calculated and visualised:

- **Total Revenue:** Sum of all pizza sales revenue  
- **Total Orders:** Total number of unique orders  
- **Total Pizzas Sold:** Total quantity sold across all orders  
- **Average Order Value (AOV):** Total Revenue / Total Orders  
- **Average Pizzas per Order:** Total Pizzas Sold / Total Orders
